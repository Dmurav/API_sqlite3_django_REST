# gamesAPI
miniproject on django rest framework 

gamesAPI - небольшое приложение, написанное с помощью django rest framework.
API позволяет вносить в базу данных информацию об играх посредством командной строки,
браузера или специализированного ПО типа POSTMAN или PyCharm. Также API позволяет
извлекать и просматривать информацию о внесённых в базу играх, редактировать эту
информацию и удалять. API предоставляет информацию в формате JSON, что позволяет 
быть приложению полноценным REST API.
